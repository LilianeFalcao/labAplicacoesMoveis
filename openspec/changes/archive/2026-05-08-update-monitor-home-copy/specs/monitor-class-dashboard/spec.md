## MODIFIED Requirements

### Requirement: Class Dashboard Entry Point Copy
O ponto de entrada para o Class Dashboard na tela Home do Monitor (o cartão da turma) deve utilizar microcopy que comunique claramente que a ação levará a um ambiente de gerenciamento de turma, e não a uma ação única de chamada.

#### Scenario: Viewing Class Action Button
- **WHEN** the monitor looks at a pending class on the Home screen
- **THEN** the action button must display the text "Acessar Turma" instead of functionally restrictive terms like "Realizar Chamada"
