## ADDED Requirements

### Requirement: Removal of Redundant Dashboard Actions
The monitor's home screen must focus on content and status rather than navigation actions that are present in the tab bar.

#### Scenario: Accessing Home Screen
- **WHEN** a monitor views the Home screen
- **THEN** only the "Turmas de Hoje" and "Próxima atividade" sections are shown, and the legacy "Ações Rápidas" grid is absent.
