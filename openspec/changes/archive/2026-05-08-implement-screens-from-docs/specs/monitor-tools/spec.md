## ADDED Requirements

### Requirement: Attendance Workflow
Monitors need a streamlined process for taking attendance and capturing activity photos.

#### Scenario: Chamada de Presença
- **WHEN** a monitor selects a group
- **THEN** they should be able to perform the check-in/out as seen in `chamada_de_presenca_monitor.png`.

#### Scenario: Activity Photos
- **WHEN** an activity is taking place
- **THEN** a monitor should be able to capture photos and write observations as in `captura_de_foto_monitor.png` and `observacoes_e_avisos_monitores.png`.
