## ADDED Requirements

### Requirement: Dashboard Summary Cards
The monitor home screen must display quick metrics at the top to provide an immediate overview of the active state.

#### Scenario: Displaying Metrics
- **WHEN** a monitor opens the home screen
- **THEN** two summary cards are shown: one for "Turmas Ativas" (total count) and one for "Presença Média" (percentage).
- **AND** each card contains a distinct icon and background color (Blue/Green) as per the design reference.
