## ADDED Requirements

### Requirement: Streamlined Dashboard Focus
The parent home screen must prioritize information about children and school announcements by removing redundant action grids.

#### Scenario: Simplified Home Navigation
- **WHEN** a parent opens the home screen
- **THEN** the "MEUS FILHOS" and "AVISOS RECENTES" sections are displayed prominently.
- **AND** the "ACESSO RÁPIDO" section is no longer visible, simplifying the vertical scrolling experience.
