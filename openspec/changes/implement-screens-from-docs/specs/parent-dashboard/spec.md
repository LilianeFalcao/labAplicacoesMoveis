## ADDED Requirements

### Requirement: Parent Home View
The home screen for parents should summarize their children's activities and provide quick navigation.

#### Scenario: Home Dos Pais
- **WHEN** a parent logs in
- **THEN** they should see their children's summary, current check-in status, and recent alerts as in `home_dos_pais.png`.

#### Scenario: Detalhes Filho
- **WHEN** a parent clicks on a child's profile
- **THEN** they should see detailed daily logs and attendance history as in `detalhes_filho.png` and `historico_de_presencas.png`.
