## ADDED Requirements

### Requirement: Management Dashboard
Administrators need to manage system entities like monitors, classes, and linking parents to children.

#### Scenario: Gestão de Monitores
- **WHEN** an administrator views the management dashboard
- **THEN** they should see the lists and management options as in `gestao_de_monitores_admin.png` and `gestao_de_turmas_admin.png`.

#### Scenario: Vincular Pais e Filhos
- **WHEN** a parent creates an account and needs to be linked to their child
- **THEN** the admin should be able to link them through the interface in `vincular_pais_e_filhos_admin.png`.
