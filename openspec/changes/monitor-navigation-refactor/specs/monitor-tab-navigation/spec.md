## ADDED Requirements

### Requirement: Expanded Monitor Tab Bar
The monitor navigation must include dedicated tabs for core activities beyond the home screen.

#### Scenario: Switching to Photo Capture from Tab
- **WHEN** a monitor taps the "Fotos" tab in the bottom navigation bar
- **THEN** the Photo Capture screen is displayed directly.

#### Scenario: Switching to Notices from Tab
- **WHEN** a monitor taps the "Avisos" tab in the bottom navigation bar
- **THEN** the Observations screen is displayed directly.
